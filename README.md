# Online Calculator
run `$ python calculator.py`

When the script is running, point your browser to:

http://localhost:8080/

### Operations
- Addition
- Subtraction
- Multiplication
- Division

### Examples
http://localhost:8080/multiply/3/5  => 15
http://localhost:8080/add/23/42     => 65
http://localhost:8080/divide/6/0    => HTTP "400 Bad Request"
